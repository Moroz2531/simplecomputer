# **Simple Computer**
## Описание
> Simple Computer - это модель простейшего компьютера, способная выполнять весь функционал настоящих ЭВМ ЭВМ. Вы можете работать с моделью как на высоком уровне, реализуя исходный код для исполнения на Simple Assembler, так и осуществлять низкоуровневое взаимодействие с ЭВМ в интерактивном режиме модели с помощью её пользовательского интерфейса, вручную вмешиваясь в процессы работы ЭВМ и изменяя состояние памяти и регистров.
## Быстрый старт
> [!TIP]  
> Убедитесь, у вас установлена система автоматизации сборки `make`. Если она отсутствует - установите её выполнив в терминале команду
> ```
> $ sudo apt install make
> ```

> 1. Склонируйте репозиторий или скачайте и распакуйте его в нужной вам директории.
> 2. Перейдите в корневой каталог проекта и соберите его, выполнив команду 
> ```
> $ make
> ```
> 3. Запустите виртуальную машину, выполнив команду
> ```
> $ ./console/pr06
> ```
> Или вы также можете запустить модель с вашим собственным шрифтом, передав при запуске в качестве аргумента путь к бинарному файлу с ним
> ```
> $ ./console/pr06 ./console/font.bin
> ```
